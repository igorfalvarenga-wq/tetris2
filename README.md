# Tetris Mobile Web

Um jogo de Tetris clássico desenvolvido com tecnologias web modernas (HTML5, Tailwind CSS, JavaScript), focado na experiência mobile.

## Funcionalidades

*   **Design Responsivo**: Interface adaptada para telas de celular.
*   **Controles de Toque**: Botões grandes para Esquerda, Direita, Girar e Descer.
*   **Mecânica de Aceleração**: Segure o botão "DESCER" para acelerar a queda da peça (Soft Drop).
*   **Sistema de Pontuação**: Acompanhe sua pontuação em tempo real.

## Como Jogar

1.  Acesse o link do jogo.
2.  Use os botões na tela para controlar as peças.
3.  Evite que as peças toquem o topo da tela!

**Arquivo Principal (Mobile):** `tetris-mobile.html`
